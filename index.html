Question Answering System

We design a question answering system similar to StackOverflow.com. The system contain regular users, moderators with various privileges, questions, answers, comments, question tags. Users gain reputation based on how good their questions and answers are. Questions, answers and comments are voted by users. Users can follow multiple tags so that whenever a new question with some tag is asked, it is shown to the followers of that tag.
